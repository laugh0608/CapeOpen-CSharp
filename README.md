## CapeOpen-NeqSimExcel

NeqSim 是一个用于计算流体行为、相平衡和过程模拟的库。该项目是 NeqSim 的 Excel 和 CAPE-OPEN 用户界面实现，可以用于 HYSYS, PROII 和 Unisim，原作者为 equinor，仓库地址：

https://github.com/equinor/neqsimExcelCapeOpen
